PayerLens

Insurance Policy Intelligence Platform

Transforming Drug Coverage Access with AI-Powered Policy Search & Citations

The Problem

💔 Insurance drug policies are scattered across many payer websites and PDFs

📝 Each payer writes rules differently with dense, technical language

🔄 Policies update frequently without clear change tracking

⏱️ Teams waste hours manually reading documents and updating spreadsheets

Result: Delayed access decisions, missed coverage opportunities, compliance risks

Our Solution

✨ A unified policy intelligence platform that:

  • Ingests policies from multiple payers automatically
  • Normalizes rules into standardized, searchable data
  • Answers coverage questions with source citations
  • Compares plans side-by-side instantly
  • Tracks policy changes and versions
  • Converses via voice call agent for accessibility

Key Features

🔍 Smart Search

Hybrid keyword + semantic search with policy metadata filters

📋 Coverage Check

Real-time answers with source citations and confidence scores

⚖️ Plan Comparison

System Architecture

End-to-end intelligent policy platform

📥 Ingestion Pipeline: PDF upload → Parse → Extract rules → Normalize → Store

🧠 AI Brain: 6 specialized agents (extraction, Q&A, compare, diff, voice, source discovery)

🗄️ Data Layer: PostgreSQL (structured) + Qdrant (embeddings/semantic search)

🎯 Query Path: User question → Retriever (hybrid) → Answer engine → Citations

Technology Stack

Python + FastAPI
Next.js + TypeScript
PostgreSQL
